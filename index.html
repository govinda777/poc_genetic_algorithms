Snake Game Genetic Algorithm POC

Prova de Conceito para Algoritmos Genéticos aplicados ao treinamento de AIs em jogos

Sobre o Projeto

Este é um projeto de prova de conceito (POC) para estudar algoritmos genéticos aplicados ao treinamento de AIs em jogos. A ideia central é treinar agentes capazes de jogar (usando as teclas: Cima, Baixo, Esquerda, Direita) através de técnicas evolutivas, inspiradas no vídeo Rede Neural aprendendo a jogar o jogo da cobrinha (SNAKE).

O projeto é dividido em três componentes principais:

  • Jogo da Cobrinha: Implementação do clássico jogo Snake em HTML/JavaScript.
  • Algoritmo Genético: Implementação em Python para treinar redes neurais a jogar o jogo.
  • Dashboard: Interface para visualizar o progresso do treinamento e os resultados.

Jogo da Cobrinha

Jogue o clássico jogo da cobrinha ou veja a IA treinada em ação. O jogo foi implementado com HTML5 Canvas e JavaScript.

Características:

  • Controles com as setas do teclado
